В рамках анализа необходимо выбрать модель, прогнозирующую вероятность продолжения деятельности стартапа (успешности стартапа), обосновать факторы, влияющие на существование стартапов. Анализ проводится по двум датасетам с тренировочными и тестовыми данными о стартапах (их названия, виды деятельности, даты создания и финансирования, географические данные, статус, продолжительность деятельности).
План анализа:
1) загрузка и получение общей информации о датасетах;
2) работа с пропусками, дубликатами, типами данных (выявление, обоснование сохранения/удаления, удаление), созданием новых столбцов;
3) исследовательский анализ: визуализация посредством графиков каждого признака из датасетов, поиск закономерностей, предварительные выводы по признакам;
4) корреляционный анализ: установление корреляции признаков (входных) с целевым (статус), построение графиков, выбор признаков для работы с моделями;
5) работа с пайплайнами (констукторами) для прогноза продолжения стартапом своей деятельности: разделение данных на выборки, подготовка данных, обучение, проверка качества; выбор лучшей модели, применение метрики;
6) визуализация важности входных признаков, рекомендации для заказчика;
7) прогноз статуса стартапа;
8) выделение признаков, влияющих на продолжение стартапом своей деятельности;
9) итоговые выводы.

Для работы использовались модели классификации: логистической регрессии, метод ближайших соседей, метод опорных векторов, дерево решений случайный лес. Лучшей оказалась модель случайного леса.

Итоговые выводы
1. В рамках анализа решалась задача прогнозирования статуса стартапов (действующие, закрывшиеся). Работа проводилась по двум датасетам.
2. В датасетах содержалось много пропусков и уникальных значений. Пропуски в финансировании заполнялись медианными значениями (в том числе отдельно для действующих и закрывшихся стартапов), пропуски в иных признаках в основном заменялись на Vague, когда по имевшимся данным нельзя было заполнить иными значениями. Например, так было со всеми кодами стран, поскольку пропуски в кодах стран соответствовали одновременному наличию пропусков в кодах регионов и названиях регионов и городов. Также была проведена работа по укрупнению кодов стран путем объединения их в 11 географических зон. Работа по укрупению проведена также с навазниями категорий (уникальные названия заменялись на первое слово в названии). Визуализация данных позволила установить, что в датасетах значения признаков схожи. Дополнительный анализ показал, что меньше всего закрывшихся стартапов в Азиатском регионе, закрывшиеся стартапы имеют меньше раундов финансирования (значимо сокращение таких компаний, выходящих на четвертый и последующие раунды финансирования), само финансирование закрывшихся стартапов меньше и такие компании в отличие от действующих стартапов меньше работают в сфере биотехнологий.
3. При корреляционном анализе была отмечена многочисленная мультиколлинеарность входных признаков. В датасеты были включены дополнительные признаки: финансирование на один раунд (среднее значение), число дней между созданием и первым финансированием, столбцы "год-месяц" для дат создания и финансирования. На мультиколлинеарность это не повлияло (при этом часть входных признаков не имели повышенных коэффициентов - более 0,9) и при обучении моделей использовались различные сочетания признаков.
4. По итогам лучшей моделью оказалась модель случайного леса для задачи классификации, метрика f1 на тренировочной выборке 0,9743, на тестовой - 0,9744, порог 0,52 обеспечивает наибольшее значение данной метрики.
5. Метод SHAP показал, что наибольшее влияние на целевой признак - статус компании - оказывает продолжительность ее деятельности, что, однако, не актуально для прогноза по недавно созданным стартапам. Для таких случаев можно использовать полученные в ходе анализа результаты о влиянии на статус географической зоны деятельности стартапа, сферы его деятельности.
6. Дополнительно изучили влияние на статус компаний новых введенных признаков. Установлено, что действующие стартапы дольше ожидают финансирование, чем закрывшиеся стартапы, получают не намного больше денег на раунд финансирования. Дополнительно по азиатскому региону где меньше всего закрывшихся стартапов, установили, что концентрация категорий видов деятельности здесь низкая, что дает основания для вывода о большем влиянии конкуренции на статус компании, нежели чисто финансовых факторах. Также определили, что для США наличие одного раунда финансирования может одинаково положительно и отрицательно повлиять на функционирование стартапа. Определили, что продолжительность действия стартапа более четырех лет может влиять на его дальнейшее функционирование. Представляется, что набор входных признаков должен быть дифференцирован, так как в датасетах явно присутствуют компании с разным периодом действия, ряд из которых трудно отнести к стартапам, и это оказывает влияние на прогнозные значения.
